# Grok Imagine Prompt Template for Daily Readings Banners

## Base Style Prompt (use for all days)

```
Photorealistic atmospheric spiritual banner photograph for a contemplative Christian website, cinematic composition, natural or dramatic lighting, meaningful mood, high detail, suitable as a website hero image with dark gradient overlay at the bottom third, 2:1 aspect ratio, professional photography aesthetic, avoid cartoonish or overly stylized looks
```

## How to Use

1. Start with the **Base Style Prompt**.
2. Append a **Day-Specific Modifier** based on the actual content of the reading.
3. Add any desired mood, time of day, or symbolic elements.
4. Optionally add: `--ar 2:1` or `--ar 16:9` depending on the tool.

## Example Prompt Structure

```
[Base Style Prompt], [specific subject or scene], [lighting / atmosphere], [symbolic elements if relevant], serene yet powerful
```

## Tips for Better Results

- Be specific about **mood** and **lighting**.
- Mention **symbolic elements** rather than forcing literal religious imagery unless it fits naturally.
- For family/personal days, lean toward warm, human, heritage photography.
- For prophetic/dramatic days, lean into cinematic lighting and skies.
- Generate a few variations per day and pick the strongest one.

## Post-Generation

- Crop/resize to exactly **1920 × 960**.
- Export as JPEG at 88% quality.
- Name as `day51-banner.jpg` etc.
